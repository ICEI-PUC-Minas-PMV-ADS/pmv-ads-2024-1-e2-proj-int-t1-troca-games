
# Projeto de Interface

## Diagrama de Fluxo

O diagrama abaixo tem como objetivo exibir o fluxo de interação do usuário com o sistema. Isso permite estrurar como será o desenvolvimento do sistema.
Serão 8 telas:

1) Tela inicial
2) Login
3) Cadastro
4) Jogo Selecionado
5) Carrinho
6) Lista de Desejos
7) Meus Itens
8) Perfil
