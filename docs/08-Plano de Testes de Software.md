# Plano de Testes de Software

Os testes funcionais a serem realizados no aplicativo são descritos a seguir.
 
| **Caso de Teste** | **Requisito Associado** | **Objetivo do Teste** | **Passos** | **Critério de Êxito** |
|:---:|:---:|:---:|:---:|:---:|
| CT-01 – Cadastrar perfil | RF-001 - O sistema deve permitir novos usuários se cadastrarem e efetuarem login para acessar suas funcionalidades. | Verificar se o usuário consegue se cadastrar na aplicação. | 1 - Acessar a página principal da aplicação;<br> 2 - Clicar no botão "Cadastrar";<br> 3 - Preencher os campos obrigatórios;<br> 4. Clicar no botão "Cadastrar". | As informações devem ser salvas no sistema para o cadastro ser concluído com sucesso. |
| CT-02 – Efetuar login | RF-001 - O sistema deve permitir novos usuários se cadastrarem e efetuarem login para acessar suas funcionalidades. | Verificar se o usuário consegue realizar login na aplicação. | 1 - Acessar a página principal da aplicação;<br>2 - Clicar no botão "Login";<br> 3 - Preencher os campos obrigatórios;<br> 4 - Clicar no botão "Entrar". | As informações preenchidas devem ser iguais às informações fornecidas no cadastro para o login ser concluído com sucesso. |
| CT-03 – Editar perfil | RF-002 - O sistema deve permitir que um usuário edite suas próprias informações de usuário. | Verificar se o usuário consegue editar suas informações. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Minha Conta" na barra de navegação;<br> 4 - Clicar no botão "Editar Conta";<br> 5 - Preencher o campo obrigatório;<br> 6 - Clicar no botão "Salvar". | As informações editadas devem ser salvas e modificadas no sistema. |
| CT-04 – Apagar perfil | RF-003 - O sistema deve permitir que o usuário apague sua conta. | Verificar se o usuário consegue apagar sua conta na aplicação. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Minha Conta" na barra de navegação;<br> 4 - Clicar no botão "Deletar conta";<br> 5 - Clicar no botão "Deletar". | A conta deve ser deletada do sistema. |
| CT-05 – Visualizar lista de itens disponíveis | RF-004 - O sistema deve apresentar uma lista com todos os itens disponíveis. | Verificar se os itens disponíveis estão aparecendo na aplicação. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos" na barra de navegação;<br> 4 - Clicar no botão "Disponíveis". | Uma lista com todos os itens disponíveis deve aparecer na página principal. |
| CT-06 – Visualizar item | RF-005 - O sistema deve permitir que qualquer usuário visualize todas as informações de um item selecionado na lista. | Verificar se as informações do item estão aparecendo corretamente. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos" na barra de navegação;<br> 4 - Clicar no botão "Detalhes" do item desejado. | As informações do item devem ser exibidas corretamente. |
| CT-07 – Visualizar disponibilidade de um item | RF-006 - O sistema deve permitir que um usuário reserve um item que não esteja reservado ainda. | Verificar se o usuário consegue selecionar um item que está disponível. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos" na barra de navegação;<br> 4 - Clicar no botão "Reservar" do item desejado. | Os itens reservados devem ficar como indisponíveis no sistema. |
| CT-08 – Cadastrar item | RF-007 - O sistema deve permitir que usuários cadastrem novos itens. | Verificar se o usuário consegue realizar o cadastro de itens. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Meus jogos" na barra de navegação;<br> 4 - Clicar no botão "Adicionar Jogo";<br> 5 - Preencher os campos obrigatórios;<br> 6 - Clicar no botão "Adicionar". | As informações devem ser salvas no sistema para o cadastro do item ser concluído com sucesso. |
| CT-09 – Editar item | RF-008 - O sistema deve permitir que um usuário atualize informações dos seus itens. | Verificar se o usuário consegue editar informações dos itens cadastrados. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Meus itens" na barra de navegação;<br> 4 - Clicar no ícone de editar;<br> 5 - Preencher os campos obrigatórios;<br> 6 - Clicar no botão "Editar". | As informações editadas do item devem ser salvas e modificadas no sistema. |
| CT-10 – Apagar item | RF-009 - O sistema deve permitir que um usuário apague itens da sua biblioteca. | Verificar se o usuário consegue apagar os itens cadastrados. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Meus itens" na barra de navegação;<br> 4 - Clicar no ícone de deletar;<br> 5. Clicar no botão "Deletar". | O item deve ser deletado do sistema. |
| CT-11 – Adicionar aos favoritos | RF-010 - O sistema deve permitir que o usuário adicione um item à sua lista de favoritos. | Verificar se a lista de favoritos do usuário fica salva no sistema. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Clicar no item desejado;<br> 4 - Clicar no botão "Favoritar";<br> 5 - Acessar a página "Meus favoritos" na barra de navegação. | Os itens selecionados devem aparecer corretamente na página "Meus favoritos". |
| CT-12 – Filtro de pesquisa | RF-011 - O sistema deve permitir que o usuário busque por itens. | Verificar se o filtro da barra de pesquisa está funcionando corretamente. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Escrever o nome do item na barra de pesquisa. | O filtro de pesquisa deve funcionar corretamente. |
