# Plano de Testes de Software

Os testes funcionais a serem realizados no aplicativo são descritos a seguir.
 
| **Caso de Teste** | **Requisito Associado** | **Objetivo do Teste** | **Passos** | **Critério de Êxito** |
|:---:|:---:|:---:|:---:|:---:|
| CT-01 – Cadastrar perfil | RF-001 - O sistema deve permitir novos usuários se cadastrarem. | Verificar se o usuário consegue se cadastrar na aplicação. | 1 - Acessar a página principal da aplicação;<br> 2 - Clicar no botão "Cadastrar";<br> 3 - Preencher os campos obrigatórios;<br> 4. Clicar no botão "Cadastrar". | As informações devem ser salvas no sistema para o cadastro ser concluído com sucesso. |
| CT-02 – Efetuar login | RF-002 - O sistema deve permitir que os usuários efetuarem login para acessar suas funcionalidades. | Verificar se o usuário consegue realizar login na aplicação. | 1 - Acessar a página principal da aplicação;<br>2 - Clicar no botão "Login";<br> 3 - Preencher os campos obrigatórios;<br> 4 - Clicar no botão "Entrar". | As informações preenchidas devem ser iguais às informações fornecidas no cadastro para o login ser concluído com sucesso. |
| CT-03 – Editar perfil | RF-003 - O sistema deve permitir que um usuário edite suas próprias informações. | Verificar se o usuário consegue editar suas informações. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Clicar no seu nome de usuário para acessar a página "Minha Conta";<br> 4 - Clicar no botão "Editar";<br> 5 - Preencher o campo obrigatório;<br> 6 - Clicar no botão "Salvar". | As informações editadas devem ser salvas e modificadas no sistema. |
| CT-04 – Apagar perfil | RF-004 - O sistema deve permitir que o usuário apague sua conta. | Verificar se o usuário consegue apagar sua conta na aplicação. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Clicar no seu nome de usuário para acessar a página "Minha Conta";<br> 4 - Clicar no botão "Deletar";<br> 5 - Clicar no botão "Deletar" para confirmar. | A conta deve ser deletada do sistema. |
| CT-05 – Visualizar lista dos itens | RF-005 - O sistema deve apresentar uma lista com todos os itens. | Verificar se todos os itens estão aparecendo na aplicação. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos" na barra de navegação. | Uma lista com todos os itens cadastrados deve aparecer na página principal. |
| CT-06 – Visualizar item | RF-006 - O sistema deve permitir o usuário visualize todas as informações de um item. | Verificar se as informações do item estão aparecendo corretamente. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos" na barra de navegação;<br> 4 - Clicar no botão "Detalhes" do item desejado. | As informações do item devem ser exibidas corretamente. |
| CT-07 – Filtro de pesquisa | RF-007 - O sistema deve permitir que o usuário busque por itens. | Verificar se o filtro da barra de pesquisa está funcionando corretamente. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos" na barra de navegação;<br> 4 - Escrever o nome do item na barra de pesquisa. | O filtro de pesquisa deve funcionar corretamente. |
| CT-08 – Reservar item | RF-008 - O sistema deve permitir que um usuário reserve um item que não esteja reservado ainda. | Verificar se o usuário consegue reservar um item que está disponível. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos" na barra de navegação;<br> 4 - Clicar no botão "Detalhes" do item desejado;<br> 5 - Clicar no botão "Reservar";<br> 6 - Clicar no botão "Reservar Jogo" para confirmar. | Os itens reservados devem ficar como indisponíveis no sistema e aparecer na página de "Jogos Reservados". |
| CT-09 – Cancelar reserva | RF-009 - O sistema deve permitir que um usuário cancele a reserva de um item. | Verificar se o usuário consegue cancelar a reserva de um item. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos Reservados" na barra de navegação;<br> 4 - Clicar no botão "Detalhes" do item desejado;<br> 5 - Clicar no botão "Cancelar Reserva";<br> 6 - Clicar no botão "Cancelar Reserva" para confirmar. | Os itens cancelados devem ficar como disponíveis no sistema e desaparecer da página de "Jogos Reservados". |
| CT-10 – Cadastrar item | RF-010 - O sistema deve permitir que usuários cadastrem novos itens. |  Verificar se o usuário consegue realizar o cadastro de itens. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Meus jogos" na barra de navegação;<br> 4 - Clicar no botão "Adicionar novo Jogo";<br> 5 - Preencher os campos obrigatórios;<br> 6 - Clicar no botão "Adicionar Jogo". | As informações devem ser salvas no sistema para o cadastro do item ser concluído com sucesso. |
| CT-11 – Editar item | RF-011 - O sistema deve permitir que um usuário atualize informações dos seus itens. | Verificar se o usuário consegue editar informações dos itens cadastrados. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Meus Jogos" na barra de navegação;<br> 4 - Clicar no botão "Detalhes" do item desejado;<br> 5 - Clicar no botão "Editar";<br> 6 - Preencher os campos obrigatórios;<br> 7 - Clicar no botão "Salvar". | As informações editadas do item devem ser salvas e modificadas no sistema. |
| CT-12 – Apagar item | RF-012 - O sistema deve permitir que um usuário apague itens da sua biblioteca. | Verificar se o usuário consegue apagar os itens cadastrados. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Meus jogos" na barra de navegação;<br> 4 - Clicar no botão "Detalhes" do item desejado;<br> 5 - Clicar no botão "Deletar";<br> 5. Clicar no botão "Deletar Jogo" para confirmar. | O item deve ser deletado do sistema. |
| CT-13 – Adicionar desejo | RF-013 - O sistema deve permitir que o usuário adicione um item à sua lista de desejos. |Verificar se o usuário consegue adicionar um item da sua lista de desejos. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Jogos" na barra de navegação;<br> 4 - Clicar no botão "Detalhes" do item desejado;<br> 5 - Clicar no botão "Adicionar à lista de desejos";<br> 6 - Acessar a página "Meus Desejos" na barra de navegação. | Os itens adicionados devem aparecer corretamente na página "Meus Desejos". |
| CT-14 – Deletar desejo | RF-014 - O sistema deve permitir que o usuário apague um item da sua lista de desejos. | Verificar se o usuário consegue apagar um item da sua lista de desejos. | 1 - Acessar a página principal da aplicação;<br> 2 - Realizar o login na aplicação;<br> 3 - Acessar a página "Meus Desejos" na barra de navegação;<br> 4 - Clicar no botão "Deletar" do item desejado;<br> 5 - Clicar no botão "Deletar da lista de desejos" para confirmar. | Os itens deletados devem desaparecer da página "Meus Desejos". |
